# 📧 Smart Email Writer Assistant (Chrome Extension)

A Chrome extension that helps users generate intelligent email replies instantly using AI.  
It combines a **Material UI-based frontend** with a **Spring Boot backend** to deliver fast and structured email responses.

---

## 🚀 Features

- ✨ Generate smart email replies in seconds
- 🎯 Multiple tone selection (formal, casual, professional, etc.)
- ⚡ Seamless integration with browser (Chrome Extension)
- 🎨 Clean and responsive UI using Material UI
- 🔗 Backend powered by Spring Boot for processing requests

---

## 🏗️ Tech Stack

### Frontend
- React.js
- Material UI (MUI)
- Axios (for API calls)

### Backend
- Spring Boot
- REST APIs

### Extension
- Chrome Extension (Manifest V3)
